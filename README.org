* Imx6ull Dockerize Compiler

** 使用

#+BEGIN_SRC text
   这个编译器是配合工程的dokcer-compse使用的;一般情况下不要直接使用.
#+END_SRC

** 编译
#+BEGIN_SRC shell
docker build -t mengdemao/bytebox-imx6ull:latest .
#+END_SRC

** 运行

#+BEGIN_SRC shell
docker run -it --rm -v $(shell pwd):/playground mengdemao/bytebox-imx6ull:latest
#+END_SRC
